# **App Name**: Dror Teper Portfolio

## Core Features:

- Dynamic CV Display: Parse data fields like work experience, education, and skills to create visually appealing CV sections
- Interactive Timeline: Display key experiences and projects in an interactive timeline format to showcase career progression.
- Skills Graph: Visualize skills proficiency using a graph or chart to highlight expertise in cyber security, forensics, and other areas.
- AI-Powered CV Enhancement Tool: Use an AI tool that, based on the user's previous roles and experience, suggests keywords that improve the CV's relevance to cyber-security related job requirements. The LLM should decide when it should suggest each piece of information.

## Style Guidelines:

- Primary color: Deep navy blue (#2E3148) to convey professionalism and trustworthiness in the cybersecurity domain.
- Background color: Light gray (#F0F2F5), almost white, to ensure readability and a clean interface.
- Accent color: Teal (#34A3A3), to add a touch of modernity and highlight interactive elements. Do not overuse teal. Limit teal to one interactive element per screen or view.
- Font pairing: 'Inter' for body text (sans-serif) and 'Literata' for headings (serif). The pairing combines readability with a touch of elegance.
- Use clean, modern icons related to cyber security and development, ensuring they are consistent in style.
- Employ a clean and structured layout with clear sections for different aspects of the CV (experience, skills, education).
- Subtle animations and transitions for interactive elements to improve user experience and engagement.
